## 💼 Sobre o Projeto

> 💡 **Projeto real aplicado em contexto acadêmico e institucional**

O **PEP — Prontuário Eletrônico do Paciente** é um sistema em desenvolvimento que será utilizado por **alunos da graduação em Psicologia** da **Faculdade de Ciências Médicas da Santa Casa de São Paulo**, no contexto do **Serviço-Escola**.

Atualmente, o projeto encontra-se em sua **fase inicial**, com foco exclusivo no **front-end**, estruturação do prontuário e experiência do usuário.

🔧 **Ainda não há:**
- Backend  
- Banco de dados  
- Integrações via API  

📈 Porém, o projeto possui **roadmap definido** para evolução gradual até sua implementação completa prevista para **dezembro de 2026**.
